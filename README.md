# Journey to Master C++ Data Structures & Algorithms

![C++ Logo](./cpp-logo.svg)

Welcome to your third-semester C++ Data Structures & Algorithms playground. This repository is designed as a focused journey to help you master DSA concepts using C++. Whether you're preparing for exams, interviews, or personal growth, follow the roadmap and exercises to build strong problem-solving skills.

## Why this repo?
- Hands-on practice with C++ and core DSA topics.
- Clean, iterative exercises: from arrays to graphs.
- Emphasis on both theory and implementation.

## Roadmap
1. Basics & STL
   - C++ fundamentals refresher
   - Standard Template Library (vector, list, map, set)
2. Linear Data Structures
   - Arrays, Linked Lists, Stacks, Queues
3. Trees
   - Binary Trees, BSTs, AVL, Segment Trees
4. Graphs
   - Representations, BFS/DFS, Shortest paths
5. Sorting & Searching
   - QuickSort, MergeSort, Binary Search
6. Greedy, DP & Advanced Topics
   - Dynamic Programming
   - Greedy algorithms
   - Tries, Heaps, Disjoint Set

## Study Plan
- Week 1-3: Fundamentals & STL
- Week 4-6: Linear Structures & Trees
- Week 7-9: Graphs & Algorithms
- Week 10-12: Advanced Topics & Project

## How to use this repo
- Browse `TASK1-2` and `TASK3` folders for practice files and previous labs.
- Add solutions in clearly named files with comments and sample input/output.
- Keep solutions efficient and well-tested.
